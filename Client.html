Class: Client

Client

A wrapper for the Asana API which is authenticated for one user

new Client(dispatcher, options, clientId, clientSecret, redirectUri, asanaBaseUrl)

Constructs a Client with instances of all the resources using the dispatcher. It also keeps a reference to the dispatcher so that way the end user can have access to it.
Parameters:
Name Type Argument Description
dispatcher Dispatcher The request dispatcher to use
options Object Options to configure the client
clientId String <optional>
 ID of the client, required for Oauth
clientSecret String <optional>
 Secret key, for some Oauth flows
redirectUri String <optional>
 Default redirect URI for this client
asanaBaseUrl String <optional>
 Base URL for Asana, for debugging
