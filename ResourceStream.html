Class: ResourceStream

ResourceStream

new ResourceStream(dispatcher, promise, dispatchOptions)

A ResourceStream is a Node stream implementation for objects that are fetched from the API. Basically, any promise for a collection of resources from the API can be wrapped in this stream, and the stream will fetch new pages of items as needed.
Parameters:
Name Type Description
dispatcher Dispatcher Dispatcher to use for requests.
promise Promise Promise for initial collection request.
dispatchOptions Object Dispatcher options to use for requests.
